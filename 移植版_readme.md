アクアノートキャンパス v2.00（現代ブラウザ動作用) 移植版 README

概要：
本ソフトは、2002年にPleiades Company 牧尾様が開発したアドベンチャーゲーム「アクアノートキャンパス v2.00」を、
Windows 11環境の現代ブラウザ（Chrome対応済み、Edge/Firefox未対応）で動作するように改造したものです。
現代ブラウザ環境で起動するため、localhost（自分のPC内）で配信して動かします。

起動方法

・start_tab.bat をダブルクリック
・黒い画面（cmd）が起動します。ゲーム実行中は閉じないでください。
・自動でブラウザが開き、ゲームが起動します。

ブラウザが開かない場合は、cmd内に表示されるURLをブラウザに貼り付けてください。

終了方法
・タイトル画面の「終了」またはブラウザを閉じてください。
・通常は自動でローカルサーバも停止します。
・停止しない場合は、表示されているcmdウィンドウを閉じてください。

推奨動作環境：
AMD ryzen7 5700x以上を推奨します。
